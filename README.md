# BookManagementAPI

### Steps:
- `npm install`
- `node index.js`

### API Endpoints:
- `/register` - POST
- `/login` - POST
- `/books` - POST
- `/books` - POST (with query as well to filter)
- `/book/:id` - PUT
- `/book/:id` - DELETE

### Libraries:
- `zod` - input validation
- `jwt` - authorization
- `bcrypt` - hashing password
- `mongoose`, `express`
